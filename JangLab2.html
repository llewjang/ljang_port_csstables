Table Terms and Definitions

How to define a table:
A structured set of data in rows and columns; it allows you to easily look up values that indicate some sort of connection. Use <table> tags to create a table.
  • <tr> use table row element when making rows in a table; nest in the table
  • <th> use table header element when writing headers for the table; use in the first row of the table containing header information
  • <td> use table data element when putting in table content in each data cell; nest in the rows
  • <thead> use when grouping header content in an HTML table
  • <tbody> use when grouping body content in an HTML table
  • <tfoot> use when grouping footer content in an HTML table
  • <caption> use when defining a table caption, insert right after the <table> tag
Difference between colspan and rowspan:
Colspan is an attribute that merges columns, allowing a single table cell to span the width of more than one cell or column. Rowspan is an attribute that merges rows, allowing a single table cell to span the height of more than one cell or row.

Difference between cell spacing and padding:
Cell spacing is the space between each cell while cell padding is the space between the cell border and the content within it.

How do you define table borders using CSS?
Define table borders with a "border" property with <table>, <th>, and <td> elements.

How do you control border widths using CSS?
Control border widths with a "border-width" property and set to a particular size with in, px, pt, cm, em, etc. For a single border, use the "border-collapse" property and the "collapse" value.
